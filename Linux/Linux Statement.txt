ls - List: liệt kê nội dung (file và thư mục)
ls -al: Liệt kê nội dung thư mục bao gồm cả file ẩn
ls -l: 
ls -a:

checkpin: powercfg/batteryreport

mkdir - Make Directory: tạo một thư mục mới

pwd - Print Working Directory: in ra đường dẫn đầy đủ đến thư mục hiện hành

cd - Change Directory

rmdir - Remove Directory: xóa một thư mục

rm <file/folder cần xóa> - Remove: xóa file, thư mục
Nếu trong folder mình xóa có data thì phải confirm lại

rm -r: xóa thư mục và toàn bộ dữ liệu trong thư mục đó

cp <file copy> <vị trí đích> - Copy: sao chép file từ vị trí nguồn đến vị trí đích

cp -r <file cần copy>: để sao chép thư mục và toàn bộ dữ liệu bên trong vào folder hiện tại

mv <địa chỉ file/folder cần move> - Move: di chuyển một file/folder bất kì đến thư mục hiện tại

cat <file cần mở> – concatenate and print files: đọc và in nội dung của file ra màn hình

tail <file cần mở> – print TAIL: đọc và in ra nội dung 10 dòng cuối cùng của file (mặc định).

tail -n <file cần mở>: n để chỉ định in n dòng ra màn hình.

less <file cần mở>: in nội dung theo từng trang
=> Dùng Ctrl+F để chuyển trang tiếp theo và Ctrl+B để chuyển về trang trước.

grep <chuỗi> <file cần mở>: tìm kiếm nội dung của file theo chuỗi cung cấp.
grep -r: để tìm kiếm trong toàn thư mục (chưa rõ)
grep -i: để tìm kiếm không phân biệt hoa thường
